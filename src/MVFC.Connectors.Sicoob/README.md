# MVFC.Connectors.Sicoob

Projeto de conector para integração com serviços do Sicoob, permitindo consultas a informações bancárias, financeiras e operacionais do sistema Sicoob.

## Funcionalidades
- Consulta de dados de cooperativas e agências do Sicoob.
- Acesso a informações financeiras, produtos e serviços oferecidos pelo Sicoob.
- Integração facilitada com APIs públicas e privadas do Sicoob.

## Uso
Inclua este projeto como dependência para acessar dados e serviços do Sicoob em aplicações .NET.

## Requisitos
- .NET SDK +9

# MVFC.Connectors.BrasilApi

Projeto responsável por integrações com a BrasilAPI, fornecendo conectores para diversos serviços públicos brasileiros.

## Funcionalidades
- Consulta de bancos, CEPs, CNPJs, clima, feriados, FIPE, IBGE, ISBN, entre outros.

## Uso
Inclua este projeto como dependência para acessar dados públicos via BrasilAPI.

## Requisitos
- .NET SDK +9

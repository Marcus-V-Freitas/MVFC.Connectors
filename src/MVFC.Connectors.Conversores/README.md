# MVFC.Connectors.Conversores

Projeto de conversores auxiliares para transformação de dados entre diferentes formatos e domínios.

## Funcionalidades
- Conversão de tipos e estruturas de dados utilizadas nos conectores MVFC.

## Uso
Utilize este projeto para facilitar a interoperabilidade entre APIs e domínios de dados.

## Requisitos
- .NET SDK

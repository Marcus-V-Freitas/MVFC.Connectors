# MVFC.Connectors.Commons

Projeto com componentes, utilitários e modelos comuns utilizados pelos demais conectores MVFC.

## Funcionalidades
- Classes e métodos compartilhados entre os projetos da solução.
- Facilita a reutilização de código e padronização.

## Uso
Inclua este projeto como dependência para acessar funcionalidades comuns aos conectores MVFC.

## Requisitos
- .NET SDK +9
